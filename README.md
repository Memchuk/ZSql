# ZSql
[В РАЗРАБОТКЕ]
ВАЖНО! Перед использованием создайте конфиг файл по пути: "{приложение.exe}\config\ZSql.data".
.data можно открыть в блокноте, это простой json.
В него надо вписать данные для подключения в виде:
{
	"Server":"0.0.0.0",
	"Port":"3306",
	"DataBase":"DB name",
	"Login":"Your login",
	"Password":"Your pass"
}
