# ZSql
### [В РАЗРАБОТКЕ]
ВАЖНО! Если вы не хотите каждый раз передавать конект стринг, создайте конфиг по пути: "{приложение.exe}\config\ZSql.data".    
В него надо вписать данные для подключения в виде:
```json
{
	"Server":"0.0.0.0",
	"Port":"3306",
	"DataBase":"DB name",
	"Login":"Your login",
	"Password":"Your pass"
}
